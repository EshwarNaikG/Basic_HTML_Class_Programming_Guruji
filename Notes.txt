🔹 1. What is HTML?

HTML (HyperText Markup Language) is the standard language used to create and structure web pages using elements (tags).

🔹 2. What is the difference between HTML and HTML5?
HTML: Older version with limited features
HTML5: Latest version with:
Semantic tags (<header>, <article>)
Audio/video support
Local storage
Better form controls
🔹 3. What are HTML tags?

Tags are elements used to define structure.
Example:

<p>This is a paragraph</p>
🔹 4. What is a semantic tag?

Semantic tags clearly describe their meaning.

Examples:

<header>
<footer>
<article>
<section>
🔹 5. What is the difference between div and span?
<div> → Block-level element
<span> → Inline element
🔹 6. What is the purpose of DOCTYPE?

It tells the browser which version of HTML is used.

<!DOCTYPE html>
🔹 7. What are attributes in HTML?

Attributes provide extra information to elements.

Example:

<a href="https://example.com">Visit</a>
🔹 8. What is the difference between id and class?
id → Unique (used once)
class → Can be used multiple times
🔹 9. What is the difference between block and inline elements?

Block elements

Take full width
Example: <div>, <p>

Inline elements

Take only required space
Example: <span>, <a>
🔹 10. What is an iframe?

Used to embed another webpage.

<iframe src="https://example.com"></iframe>
🔹 11. What is localStorage in HTML5?

Used to store data in browser permanently (until cleared).

localStorage.setItem("name", "Eshwar");
🔹 12. Difference between localStorage and sessionStorage?
Feature	localStorage	sessionStorage
Lifetime	Permanent	Session only
Scope	Same browser	Tab-specific
🔹 13. What is the <meta> tag?

Provides metadata (SEO, charset, viewport).

Example:

<meta charset="UTF-8">
🔹 14. What is the difference between GET and POST?
GET → Data visible in URL
POST → Data hidden (more secure)
🔹 15. What is form validation?

Ensuring user input is correct.

Example:

<input type="email" required>
🔹 16. What is the difference between HTML and CSS?
HTML → Structure
CSS → Styling
🔹 17. What is the alt attribute?

Used in images for accessibility.

<img src="image.jpg" alt="Description">
🔹 18. What are void elements?

Elements without closing tag.

Examples:

<br>
<img>
<hr>
🔹 19. What is the difference between strong and b?
<strong> → Important (semantic)
<b> → Bold (visual only)
🔹 20. What is the viewport?

Controls layout on mobile devices.

<meta name="viewport" content="width=device-width, initial-scale=1.0">
🔥 Advanced HTML Questions
21. What are data attributes?

Custom attributes:

<div data-user="123"></div>
22. What is SEO in HTML?

Optimizing webpage for search engines using:

meta tags
headings
alt attributes
23. What is canvas in HTML5?

Used to draw graphics using JavaScript.

24. What is SVG?

Scalable Vector Graphics (for icons, shapes).

25. Difference between cookies and localStorage?
Cookies → Small data, sent to server
localStorage → Large data, stored in browser
